Project Title

House Price Prediction using Linear Regression

Objective

The objective of this project is to predict house prices based on features such as area, bedrooms, bathrooms, parking, and stories using Linear Regression.

Steps Performed
Loaded and analyzed the housing dataset
Cleaned and preprocessed the data
Converted categorical data into numerical form
Performed Exploratory Data Analysis (EDA)
Visualized data using heatmaps and graphs
Selected input features and target variable
Split dataset into training and testing sets
Implemented Linear Regression model
Predicted house prices
Evaluated model using:
Mean Squared Error (MSE)
R² Score
Tools & Technologies Used
Python
Google Colab
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Outcome

The Linear Regression model successfully predicted house prices with good accuracy. The project demonstrated the relationship between housing features and price prediction.
